# Youssef Abdelaziz Portfolio

A modern React/Next.js portfolio website built for CV sharing and professional presentation.

## Included sections

- Hero
- About
- Education
- Experience
- Skills
- Projects
- Certificates
- Contact

## Included files

- CV PDF: `public/cv/youssef-abdelaziz-cv.pdf`
- DP-600 certificate PDF
- DP-700 certificate PDF

## How to run locally

```bash
npm install
npm run dev
```

Then open:

```bash
http://localhost:3000
```

## How to deploy

Recommended: Vercel

1. Upload this folder to GitHub.
2. Go to Vercel.
3. Import the GitHub repository.
4. Click Deploy.

## What to customize later

- Add your LinkedIn link when ready.
- Replace the initials card with a profile photo if you want.
- Add more projects or live project links when available.
